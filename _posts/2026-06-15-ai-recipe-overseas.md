## AIにレシピを聞くようになった

最近、料理のレシピをAIに聞いて作ることが増えた。

きっかけはふとした思いつきで、「英語でハワイ料理のレシピを調べてほしい」とAIエージェントに頼んでみたところ、思いのほか面白い体験ができた。

これまでに作ったのは、ハワイのガーリックシュリンプとマカロニサラダ、イタリアンの白ラグーソースのパスタ、そして今日はフレンチのオニオングラタンスープを仕込んでいる。

## 現地語で検索してもらう、というポイント

このAI活用法の肝は、**現地の言語でレシピを検索してもらう**ことにある。

オニオングラタンスープはフランス語で検索してもらったのだが、出てきたのは「フランス人のおばあちゃんが作っているようなレシピ」だった。日本語で「オニオングラタンスープ レシピ」と検索して出てくるものとは、明らかに雰囲気が違う。

日本で紹介されているレシピは、日本人の口に合うよう調整されていたり、日本で手に入りやすい食材に置き換えられていることが多い。それはそれで便利なのだが、どこかパンチに欠ける。

一方、現地のレシピはそのまま現地のものだ。日本人向けに丸められていない分、初めて食べる風味がする。

## 日本ではなじみのない食材が出てくる

現地レシピの面白さのひとつが、**日本ではあまり見かけない食材やスパイスが登場する**ことだ。

- **ハワイのマカロニサラダ**：マヨネーズをたっぷり使うのが特徴。日本のものとは量が段違いで、これが本場の味らしい。
- **白ラグーのパスタ**：フェンネル（ういきょう）を使う。日本ではなじみの薄いスパイスだが、これがソースに独特の甘みと清涼感を加える。
- **オニオングラタンスープ**：玉ねぎを4個使い、薄切りにして弱火で1時間ほど煮詰め、飴色になるまでじっくり炒める。スパイスはタイムとローレルが基本だが、本来はもっと現地ならではのハーブを使うらしい。

フェンネルやその他のハーブは普通のスーパーには置いていないこともあるが、富沢商店のような専門店に行けばだいたい揃えられる。

実際にAIが出してくれたレシピを以下に載せておく。

### ハワイ風ガーリックシュリンプ（4人分）

**材料**

| 材料 | 分量 |
|---|---|
| 殻付きエビ（ブラックタイガー推奨） | 400〜500g |
| にんにく | 6〜8片（みじん切り） |
| バター | 60g |
| サラダ油 | 大さじ2 |
| 薄力粉 | 大さじ4 |
| パプリカパウダー | 大さじ1 |
| 塩 | 小さじ1 |
| 黒こしょう | 少々 |
| レモン汁 | 大さじ1 |
| カイエンペッパー（大人向け） | 少々 |

**作り方**

1. **エビの下処理**：殻付きのまま背に切れ込みを入れて背ワタを取り、水気を完全に拭く。
2. **ガーリックバター**：弱火でバターを溶かし、にんにくを焦がさず2〜3分炒めて香りを出したら、別皿に取り出す。
3. **粉をまぶす**：薄力粉・パプリカ・塩・黒こしょうを混ぜ、エビに軽くまぶす。
4. **焼く**：フライパンに油を引き、中火〜強火でエビを片面2分ずつ焼く。
5. **仕上げ**：ガーリックバターを戻して全体に絡め、最後にレモン汁をかける。

**コツ**：殻付きのまま焼くことで旨味が逃げない。エビの水分は徹底的に除去。にんにくは絶対に焦がさない。白ご飯との相性が抜群。

### ハワイ風マカロニサラダ（Mac Salad）（4人分）

**材料**

| 材料 | 分量 |
|---|---|
| マカロニ | 200g |
| マヨネーズ | 200g |
| りんご酢 | 大さじ2 |
| 砂糖 | 大さじ1 |
| にんじん | 50g（細切りまたはすりおろし） |
| 玉ねぎ | 大さじ2（みじん切り） |
| 塩 | 小さじ1 |
| 黒こしょう | 少々 |
| 牛乳 | 大さじ2〜3 |

**作り方**

1. **マカロニを柔らかめに茹でる**：表示時間＋1〜2分。
2. **熱いうちに酢を絡める**：湯切り直後にりんご酢を直接絡める。
3. **ドレッシングを作る**：マヨネーズ・砂糖・塩・黒こしょう・にんじん・玉ねぎを混ぜる。
4. **合わせて冷やす**：全部混ぜて冷蔵庫で30分以上冷やす。
5. **最後に牛乳で緩める**：冷えると固くなるので、食べる前に牛乳で好みの硬さに調整する。

**ポイント**：マヨネーズ200gというのが日本人の感覚では多く感じるが、これが本場の味。りんご酢を熱いうちに絡めることでマカロニに下味がつく。

### 白ラグー × フィットチーネ（自作サルシッチャ）（2〜3人分）

**材料**

*パスタ生地*

| 材料 | 分量 |
|---|---|
| セモリナ粉 | 180g |
| 薄力粉 | 120g |
| 卵 | 3個（約150g） |
| 水 | 15g |
| 塩 | 3g |
| オリーブオイル | 5g |

*サルシッチャ*

| 材料 | 分量 |
|---|---|
| 豚ひき肉（粗挽き） | 300g |
| 塩 | 5g |
| にんにく | 1片 |
| 黒胡椒 | 多め |
| フェンネルシード（潰す） | 小さじ1 |
| オリーブオイル | 小さじ1 |

*ラグー*

| 材料 | 分量 |
|---|---|
| 玉ねぎ | 1/2個 |
| にんじん | 1/3本 |
| セロリ | 1/2本 |
| 白ワイン | 100ml |
| 水またはブイヨン | 100ml |
| オリーブオイル | 適量 |
| パルミジャーノ | 適量 |

**作り方**

1. **パスタ生地**：材料を混ぜて5〜10分こねる。ラップして常温30分→冷蔵4時間以上休ませる。使用前30分は常温に戻す。
2. **成形**：パスタマシン（Atlas150）で厚さ5〜6（約1.3〜1.5mm）に伸ばす。セモリナ粉で打ち粉をして5〜10分乾燥させてからフィットチーネにカットし、よくほぐす。
3. **サルシッチャ**：材料をすべて混ぜて2〜3分練る。冷蔵庫で30分以上休ませる。
4. **白ラグー（ソフリット）**：弱火で玉ねぎ・にんじん・セロリを色付けないよう10分以上炒める。中火にしてチリチリと音がするまで水分を飛ばす。
5. **サルシッチャを加える**：フライパンに投入し、最初は触らず焼き色をつけてからほぐす。
6. **白ワインを加える**：強火でアルコールを飛ばす。
7. **煮込む**：水またはブイヨンを加えて20〜30分煮る。
8. **仕上げ**：パスタを塩1%の湯で2〜3分茹でる。フライパンに移し、茹で汁大さじ2〜3を加えて中火で混ぜながら乳化させる。パルミジャーノとオリーブオイルで仕上げる（好みでレモンの皮またはバター少量を加えても）。

**重要ポイント**
- カット前に軽く乾燥させる（くっつき防止）
- 肉は最初触らずに焼く（メイラード反応）
- 必ずフライパンで乳化させる。茹でたパスタをそのままソースに乗せるだけはNG

### オニオングラタンスープ（4人分）

**材料**

| 材料 | 分量 |
|---|---|
| 黄玉ねぎ | 600〜750g |
| バター | 40g |
| 小麦粉 | 20〜25g |
| 辛口白ワイン | 200ml |
| 牛肉のブイヨン | 1L |
| ブーケガルニ（タイム・ローリエ） | 1束 |
| カンパーニュまたはバゲット | 4枚 |
| グリュイエールチーズ | 120g |
| 塩・こしょう | 適量 |

**作り方**

1. 玉ねぎを薄切りにする。
2. 大きな鍋でバターを溶かす。
3. 玉ねぎを加え、弱火でじっくり炒めて黄金色〜飴色にする。
4. 小麦粉を振り入れて混ぜる。
5. 白ワインを加えて鍋底の旨味をこそげ取る。
6. ブイヨンとブーケガルニを加える。
7. 約30分煮込む。
8. 塩・こしょうで味を整える。
9. パンをこんがり焼く。
10. 耐熱ボウルにスープを注ぐ。
11. パンをのせる。
12. グリュイエールチーズをたっぷりかける。
13. グリルでチーズに焼き色が付くまで焼く。
14. 熱々で供する。

**フランスのレシピで気づいたこと**：日本のレシピよりも「玉ねぎを40〜60分かけて徹底的に飴色にする」ことを重視している。チーズやパンよりも、むしろ玉ねぎのキャラメリゼが主役という考え方だ。

## AIがうまくサポートしてくれる部分

このレシピ検索でAIが特に役立つのは、次の3点だ。

1. **現地語でしか見つからない情報を日本語に翻訳してくれる**：フランス語や英語のレシピをそのまま読むのは難しいが、AIが自然な日本語に直してくれる。

2. **入手困難な食材を代替提案してくれる**：「この食材は日本で手に入らないのですが、何で代替できますか？」と聞けば、近いものを教えてくれる。完全に同じ味は出せなくても、かなり近づけることができる。

3. **どこで買えるか教えてくれる**：「このスパイスは日本のどこで買えますか？」と聞くと、富沢商店や成城石井、ネット通販など具体的な入手方法を教えてくれる。

## 海外旅行をしているような気分

出来上がった料理を食べると、日本ではあまり嗅いだことのない香りがして、なんとなく海外旅行をしているような気分になる。

ガーリックシュリンプを食べたときはハワイのノースショアのプレートランチを、白ラグーのパスタを食べたときはイタリアのどこかのトラットリアを、ぼんやりと想像した。

旅行に行かなくても、食卓でその国の空気を少し感じられる。AIを使った現地語レシピ検索は、なかなかおすすめの使い方だと思っている。
